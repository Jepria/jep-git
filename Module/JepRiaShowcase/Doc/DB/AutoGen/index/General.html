pkg_JepRiaShowcasepkg_JepRiaShowcase.body
Реализуется с помощью целей сборки модуля из make-целей Установка модуля в БД.
Функционал, являющийся примером использования компонентов, библиотек и подходов.
[+] Добавлены таблицы и интерфейсные функции для сущностей “Поставщик”, “Товар”, “Товар в магазине”, “Запрос на закупку”, “Обработка запроса” и связанные с ними справочники.
[*] Для поставщиков ( таблица jrs_supplier) добавлены дополнительные атрибуты: телефон ( phone_number), факс ( fax_number) и банковские реквизиты ( bank_bic, recipient_name, settlement_account).
[*] Для установки разделов каталога, к которому относится товар, добавлена функция pkg_JepRiaShowcase.setGoodsCatalogLink, функции createGoodsCatalogLink и deleteGoodsCatalogLink удалены.
Целиком собирает проект.
