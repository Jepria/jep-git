9.1.0 ( 2016-05-24)

[+] Добавлен скрипт по созданию пользователей itm, itm_user Install/Schema/Last/Dba/run.sql.

[+] Добавлена сущность “Запрос функционала” ( feature).

[+] Добавлен скрипт по выдаче прав на использование модуля ( Install/Grant/Last/run.sql).

8.6.0 ( 2015-08-03)

[*] Для установки разделов каталога, к которому относится товар, добавлена функция pkg_JepRiaShowcase.setGoodsCatalogLink, функции createGoodsCatalogLink и deleteGoodsCatalogLink удалены.

[*] Для получения информации о разделах каталога, к которым относится товар, в функцию pkg_JepRiaShowcase.getGoodsCatalog добавлен параметр goodsId и в возвращаемый курсор добавлены поля goods_link_flag и descendant_goods_link_flag.  Функция getGoodsCatalogLink удалена.

[*] В функцию pkg_JepRiaShowcase.getBank добавлен параметр maxRowCount.

[*] Для поставщиков ( таблица jrs_supplier) добавлен дополнительный атрибут: флаг привилегированного поставщика ( privilege_supplier_flag).

8.3.0 ( 2015-03-25)

[*] Для поставщиков ( таблица jrs_supplier) добавлены дополнительные атрибуты: телефон ( phone_number), факс ( fax_number) и банковские реквизиты ( bank_bic, recipient_name, settlement_account).

[*] Внешние ключи jrs_request_fk_goods_id и jrs_goods_fk_supplier_id пересозданы с опцией “on delete cascade”.

1.0.0 ( 2014-05-12)

[+] Добавлены таблицы и интерфейсные функции для сущностей “Поставщик”, “Товар”, “Товар в магазине”, “Запрос на закупку”, “Обработка запроса” и связанные с ними справочники.
